## Using Webkit 

```
	/// the animation
	@keyframes example {
	    from {
		    background-color: red;
		 }
	    to {
	    	background-color: yellow;
	    }
	}
	
	// element to apply the animation to
	div {
	    width: 100px;
	    height: 100px;
	    background-color: red;
	    animation-name: example;
	    animation-duration: 4s;
	}
```
